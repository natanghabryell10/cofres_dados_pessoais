# cofres_dados_pessoais
Projeto simples em Python que permite salvar dados pessoais de forma estruturada usando JSON.   Funcionalidades - Senha mestre para proteger o acesso - Adicionar dados (nome, login, senha) - Visualizar dados salvos - Salvar dados em arquivo `dados.json`  Abra `cofre.py` Digite a senha mestre  Use o menu para adicionar ou visualizar dados
